# reading
通过微信小程序来管理自己的读书记录。 后端技术栈：SpringBoot、MyBatisPlus、MySQL

## 项目架构
    SpringBoot2.7.7、MyBatisPlus、MySQL5.7.9
## 项目优势
1. 用户可以根据自己的爱好在收藏的书架上对读书进行不同方式的排版
   (加入的时间、书名、随机排列)

2. 用户可以搜索自己的读书历史


## 模块开发

    读书的操作CURD
    
    添加读书
    更新读书
    删除读书
    查询读书

    用户的操作CURD
    添加用户
    删除用户    

    用户登录与注册
    第三方登录与注册，使用微信号
